# 音乐播放器项目

## 项目简介
本项目为一个基于 HTML、CSS、JavaScript 的网页音乐播放器，支持本地音频文件播放、背景图片轮播、水波纹特效等功能。

## 主要功能
- 自动加载 music 目录下所有音频文件作为播放列表
- 支持多种播放模式（列表循环、随机播放、单曲循环）
- 背景图片自动轮播（bgimg 目录下图片）
- 水波纹动态特效
- 音量调节、进度条、隐藏/显示播放器界面

## 使用说明
1. 更换音乐：将音频文件（mp3等）放入 music 目录，并在music\musicList.js添加文件名。
2. 更换图片：将背景图片放入 bgimg 目录，并在 bgImgList.js 中添加文件名。
3. 修改参数：可通过修改 index.html 调整播放模式、音量、背景切换间隔等参数。
